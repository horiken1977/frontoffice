📁 ファイルアップロードガイド

社内データ検索で活用できるファイル形式と推奨項目

📋 基本方針

  • • どんなデータでも活用可能: 営業、マーケティング、顧客、商品など幅広く対応
  • • 形式は柔軟: CSV、Excel、PDF、テキストファイル等をサポート
  • • 項目名は自由: 既存のファイルをそのまま活用できます
  • • 日本語完全対応: 項目名、内容ともに日本語で問題ありません

📄 対応ファイル形式

✅ 推奨形式

  • 📊 CSV (.csv) - 最も検索精度が高い
  • 📈 Excel (.xlsx, .xls) - 表形式データに最適
  • 📋 Markdown (.md) - 構造化テキストに適用

🔵 対応形式

  • 📄 PDF (.pdf) - 提案書、契約書等
  • 📝 テキスト (.txt) - メモ、議事録等
  • 📋 Word (.docx) - 文書ファイル

💡 効果的な活用のコツ

📋 データ準備のポイント

  • • 項目名は分かりやすく（「顧客名」「売上」等）
  • • 数値は単位を統一（万円、億円等）
  • • 日付は統一形式で（2025/1/15等）
  • • 空欄は「未定」「不明」等で埋める

🔍 検索精度向上のコツ

  • • キーワードを複数含める（業界、課題等）
  • • 関連情報をまとめて記載
  • • 略語は正式名称も併記
  • • 重要な情報は上位項目に配置

❓ よくある質問

Q: 既存のExcelファイルをそのまま使えますか？

A: はい、可能です。項目名や形式の変更は不要で、現在お使いのファイルをそのままアップロードできます。

Q: データ量に制限はありますか？

A: Phase 1では1ファイル50MB、合計200MBまで対応しています。大容量データは複数ファイルに分割してください。

Q: どんな項目があれば良いか分からない場合は？

A: まずは現在お持ちのデータをアップロードしてみてください。足りない項目は後から追加可能です。

Q: セキュリティは大丈夫ですか？

A: Phase 1ではローカルストレージでの一時保存のみです。Phase 2以降で企業向けセキュリティ機能を追加予定です。
